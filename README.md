# 新橋料亭 お江戸 - 老舗和風料理店サイト

**伝統と上品さをまとった和食料亭のコーポレートサイト**（練習サンプル）

書籍『動くWebデザインアイディア帳』を参考に作成した、東京新橋で二百年続く老舗料亭のランディングページです。  
フルスクリーンスライダー、円形背景のメニュー展開、線が伸びる演出、ぼかし登場アニメーションなど、和の静けさと格式を感じさせる繊細な動きが特徴です。  
**完全レスポンシブデザイン**で、PCからスマホまで美しいレイアウトを保ちます。

## デモ（ライブプレビュー）
https://ryoutei-htmlcssjs-practice.vercel.app/

## 主な実装機能
- **フルスクリーンスライダー**（slick-carousel） - フェード切り替え、自動再生、レスポンシブ対応
- **ハンバーガーメニュー** - クリックで円形背景拡大 + ナビ出現、背景ぼかし効果
- **線が伸びる演出**（lineTrigger） - メニュー画像やアクセスエリアでスクロールに合わせて線が伸びる
- **ぼかし登場アニメ**（blurTrigger） - 見出しやテキストがスクロールでぼかしから現れる
- **スマホ対応完全レスポンシブ** - 画面サイズに応じてレイアウトが最適化（メニュー、画像、テキスト）
- **ページトップボタン** - スクロールで出現
※Splash Screen Animationとsvg Animationはオミット

## 使用技術・ライブラリ
- HTML5 / CSS3
- jQuery 3.4.1
- [slick-carousel](https://kenwheeler.github.io/slick/) - フルスクリーンスライダー
- Google Fonts（Noto Serif JP）
